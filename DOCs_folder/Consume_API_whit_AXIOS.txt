Que es AXIOS?

Axios es una biblioteca de cliente HTTP basada en promesas que simplifica 
el envío de solicitudes HTTP asincrónicas a puntos finales REST. 
Este punto final en nuestro caso es la API JSONPlaceholder Posts, 
a la que realizaremos solicitudes GET, POST y DELETE.
